HW12 - Find the Hub in Minimum Spanning Tree (MST)

Assignment

Construct the minimum spanning tree for given N points.
Later, find out the hub in this minimum spanning tree and report the total lenth of paths from the hub to other N-1 points. The definition of the Hub in a MST is a point in the MST with the smallest sum of the shortest paths to other N-1 points in the MST.

Descriptions

  • Please write a program (FindHub.java) that reads in N 2-dimensional points (both x and y ranges from 0 to 1).
  • The first line of the input file specifies the number of points (N), followed by the 2-dimensional coordinates of the N points.
  • Construct a minimum spanning tree for given N points.
  • Build a graph of this minimum spanning tree.
  • Find out which point is the "hub" point. The "hub" is defined as the point which has the smallest total length of the shortest paths to other N-1 points.
  • Report the total length of the shortest paths from the hub to other N-1 points
    usage: System.out.printf("%5.5f\n", d);
